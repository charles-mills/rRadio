# rRadio Localisation Key Context

Reviewed: 2026-05-11

Canonical locale source lives in `data/locales/*.json`. Runtime Lua chunks in `lua/rradio/client/lang/data_*.lua` are generated by `scripts/build_locale_artifacts.py`; do not edit those chunks by hand.

Translator notes: keep `rRadio`, `Garry's Mod`, `Lua`, `URL`, key names, numeric ranges, and placeholders such as `{shown}` and `{total}` intact. In this UI, “local” means client-side playback on the player machine. “Station” means an internet radio stream entry. “Boombox” is an in-world portable radio entity, while “vehicle radio” is the in-car radio interface. Use short game-UI phrasing where possible; button labels should stay compact.
