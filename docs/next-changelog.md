rRadio v2.0.0

- Added 30,000 new radio stations.
- Added CAMI-gated property menu toggles to make individual boomboxes public or permanent.
- Added GUI to add and manage custom radio stations.
- Added GUI to manage server-wide settings.
- Added Inter font throughout the UI.
- Added PermaProp tool compatibility.
- Added client setting to disable visibility of all resize grabbers.
- Added crossfade for station changes.
- Added keyboard controls for the radio UI.
- Added option in properties menu to copy a stream URL to the clipboard.
- Added option in properties menu to favorite or unfavorite the active radio station.
- Added property menu quick play flyout for recent stations.
- Added recently played stations view.
- Added setting to mute radios whilst in the escape menu (on by default).
- Added soft UI mode disabling borders across the radio menu.
- Added status accent to connecting / connected radio station rows.
- Added system to block 'kill' keybinds whilst in vehicles, if the key is also the menu key.
- Changed menu dragging to prevent dragging menu outside of view.
- Changed menu opening to place the cursor on the menu instead of at the screen centre.
- Changed radio UI to persist position.
- Many small UI and UX changes.
- Validated all new and existing stations expose a real audio stream.

This update is large and there may be new bugs, please report them in thread.
